# 🎓 EduSpace - Образовательный портал

**EduSpace** — современный веб-портал для студентов образовательных учреждений. Платформа предоставляет централизованный доступ к учебной информации, расписанию, оценкам, документам и другим образовательным сервисам.

## ✨ Особенности

### 🏠 Главная страница
- **Персонализированная панель** с приветствием и актуальной информацией
- **Статистические карточки** - GPA, посещаемость, выполненные задания, кредиты ECTS
- **Быстрые действия** - ссылки на основные разделы портала
- **Уведомления** - важные объявления и напоминания
- **Прогресс семестра** - визуальное отображение академического прогресса
- **Виджет погоды** - текущая погода в Алматы

### 👤 Профиль студента
- **Личная информация** - просмотр и редактирование данных
- **Академическая информация** - специальность, курс, группа
- **Достижения** - полученные награды и сертификаты
- **История активности** - журнал действий в системе
- **Смена пароля** - безопасное обновление учетных данных

### 📚 Учебный процесс
- **Расписание занятий** - интерактивное расписание с фильтрами
- **Журнал оценок** - текущие и итоговые оценки
- **Индивидуальный план** - персональная траектория обучения
- **Учебные материалы** - лекции, презентации, документы
- **Тестирование** - онлайн-экзамены и тесты
- **Апелляции** - подача апелляций по оценкам
- **Транскрипт** - официальная выписка оценок

### 📢 Коммуникации
- **Новости** - последние новости и объявления
- **Объявления** - официальные уведомления
- **События** - календарь мероприятий
- **Преподаватели** - контактная информация преподавателей

### 🏛️ Административные функции
- **Документы** - справки, заявления, дипломы
- **Финансы** - оплата обучения, стипендии
- **Настройки** - персонализация интерфейса

## 🛠️ Технологии

### Frontend
- **React 18** - современная библиотека для создания пользовательских интерфейсов
- **React Router 6** - маршрутизация SPA
- **Bootstrap 5** - адаптивная CSS-фреймворк
- **Bootstrap Icons** - иконочный шрифт
- **CSS3** - современные стили с переменными и Flexbox/Grid

### Инструменты разработки
- **Create React App** - инструментарий для быстрого старта
- **ES6+** - современный JavaScript
- **CSS Custom Properties** - CSS переменные для централизованного управления стилями
- **Responsive Design** - адаптивный дизайн для всех устройств

## 🚀 Установка

### Требования
- **Node.js** >= 16.0.0
- **npm** >= 8.0.0 или **yarn** >= 1.22.0

### Клонирование репозитория
```bash
git clone https://github.com/yourusername/eduspace.git
cd eduspace
```

### Установка зависимостей
```bash
# Используя npm
npm install

# Или используя yarn
yarn install
```

## 🖥️ Использование

### Запуск в режиме разработки
```bash
# npm
npm start

# yarn
yarn start
```

Приложение будет доступно по адресу: `http://localhost:3000`

### Сборка для продакшена
```bash
# npm
npm run build

# yarn
yarn build
```

Собранное приложение будет находиться в папке `build/`

### Запуск тестов
```bash
# npm
npm test

# yarn
yarn test
```

## 💻 Разработка

### Архитектура
Проект следует принципам:
- **Component-Based Architecture** - модульная структура компонентов
- **Separation of Concerns** - разделение ответственности
- **DRY (Don't Repeat Yourself)** - избежание дублирования кода
- **Mobile-First Design** - приоритет мобильной версии

### Структура компонентов
```javascript
// Пример структуры компонента
components/
├── ComponentName/
│   ├── index.js          # Главный файл компонента
│   ├── ComponentName.js  # Логика компонента
│   ├── styles.css        # Стили компонента
│   └── README.md         # Документация компонента
```

### CSS организация
- **CSS Custom Properties** - для централизованного управления дизайн-системой
- **BEM методология** - для именования классов
- **Mobile-First** - адаптивный дизайн от меньшего к большему экрану

### Код-стайл
- **ES6+** синтаксис
- **Функциональные компоненты** с хуками
- **Деструктуризация** объектов и массивов
- **Template literals** для строк
- **Async/await** для асинхронных операций

## 📱 Адаптивность

Проект полностью адаптивен и оптимизирован для всех устройств:

### Breakpoints
- **Mobile**: ≤ 576px (Extra small devices)
- **Small Mobile**: ≤ 400px (Very small devices)
- **Tablet**: ≤ 768px (Small devices)
- **Small Desktop**: ≤ 992px (Medium devices)
- **Large Desktop**: ≤ 1200px (Large devices)
- **Extra Large**: > 1200px (Extra large devices)

### Мобильные особенности
- **Гамбургер-меню** для навигации
- **Touch-friendly** элементы управления (минимум 44px)
- **Отзывчивая типографика** - автоматическое масштабирование шрифтов
- **Оптимизированные формы** - увеличенные поля ввода
- **Swipe gestures** - поддержка жестов
- **Landscape orientation** - поддержка поворота экрана

### Performance оптимизации0
- **Lazy loading** - отложенная загрузка изображений
- **Code splitting** - разделение кода на чанки
- **CSS optimizations** - минификация и оптимизация стилей
- **Responsive images** - адаптивные изображения
